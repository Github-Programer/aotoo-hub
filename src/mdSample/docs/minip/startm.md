# hub 的小程序

hub的小程序项目基于微信原生开发，webpack编译，hub按照1:1对等输出小程序源码，仅利用webpack处理一下小程序的一些小毛病，比如markdown，动态样式，html等等。(输出不会产生webpack的冗余代码)，支持热更新
